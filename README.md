# ATC — Algorithmic Thinking Club

Official website for the Algorithmic Thinking Club at Bonney Lake High School.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Structure

```
app/            → pages and routes
components/     → shared UI components
data/           → local content data (competitions, projects, members)
public/         → static assets
```

## Editing Content

All content lives in the `data/` folder as TypeScript files. Edit those to update what shows on the site.
